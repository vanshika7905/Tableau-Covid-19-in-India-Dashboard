🇮🇳 COVID-19 in India — Tableau Dashboard Analysis
📌 Project Overview

This project presents an interactive Tableau dashboard analyzing the impact of COVID-19 across India. The dashboard provides a comprehensive state-wise and demographic analysis of cases, deaths, testing, and vaccination progress.

The goal of this project is to transform raw COVID-19 data into meaningful insights using data visualization and storytelling techniques.

🎯 Key Insights & Features

🗺️ 1. State-wise Death Analysis

Interactive map displaying total deaths by state

Easy comparison of high and low impacted regions

📈 2. COVID Cases Trend Over Time

Time-series analysis of:

Confirmed Cases

Recovered Cases

Death Cases

Trend comparison across different time periods

👥 3. Age Group Distribution

Donut chart showing affected age groups

Helps identify most impacted age category

🚻 4. Gender Distribution

Percentage breakdown of Male vs Female cases

Clear demographic understanding

🧪 5. ICMR Testing Labs

State-wise comparison of testing labs

Highlights testing infrastructure availability

💉 6. Vaccination Analysis

First Dose vs Second Dose comparison

Vaccine-wise distribution (Covishield, Covaxin, Sputnik V)

Overall vaccination progress tracking

🧪 7. Statewise Testing Details

Comparison of total tests conducted by major states
